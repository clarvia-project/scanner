## 공통 이탈 패턴 4개

1. **"한 번 쓰고 끝" 구조** — 대시보드, 히스토리, 알림 전부 없음 → 재방문 동기 0
2. **데이터가 갇혀 있음** — 내보내기, 공유 링크, 벌크 API 없음 → 워크플로 통합 불가
3. **15회/월은 테스트용** — 어떤 페르소나도 프로덕션 불가 + 유료 플랜이 안 보임
4. **카테고리 부재** — 64개 서비스가 뒤섞여 있어 의미 있는 비교/분석 불가

## 최소 구현으로 점수 올릴 수 있는 것 (P0)

### 1. 서비스 대시보드 (6명 중 5명 요구)

- 스캔 결과 공유 링크 (clarvia.art/scan/{id})

### 2. 유료 플랜 노출 + 결제 (6명 중 4명 "돈 내고 싶은데 방법 없음")

- Team: 2000회/월, $79/월 → 박지영 전환

- 2-5개 서비스 나란히 차원별 비교

### 4. 카테고리 태그 (4명 요구)
- 리더보드 카테고리 필터
- 카테고리별 벤치마크 API
